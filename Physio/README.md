## 🚀 Comment l'utiliser

### Option 1 — Localement

Clone le repo et ouvre `index.html` dans ton navigateur :

```bash
git clone <url-du-repo>
cd Physio
# Double-clique sur index.html ou ouvre-le dans ton navigateur
```

### Option 2 — Via GitHub Pages

1. Va dans **Settings → Pages** sur GitHub
2. Source : `Deploy from a branch` → `main` / `root`
3. Ton site sera disponible à `https://<ton-pseudo>.github.io/<nom-du-repo>/`
